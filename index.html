Summary: Latitude vs. X

The purpose of this project was to create visualizations of the weather in 500+ cities across the world of varying distance from the equator, to showcase the following relationships: Temperature (F) vs. Latitude, Humidity (%) vs. Latitude, Cloudiness (%) vs. Latitude and Wind Speed (mph) vs. Latitude.

The dataset was assembled of unique cities based on latitude and longitude. A weather check was done on each of the cities using a series of API calls through the OpenWeatherMap API. The data tab provides the source data including City, Cloudiness, Country, Date, Humidity, Latitude, Longitude, Max Temp and Wind Speed. The visualizations were created as part of the analysis using Pythons Matplotlib library, each scatterplot showcasing the results can be found in the plots tab with descriptions of trends, correlations and significance. The comparisons tab also contains all of the scatterplots on the same page so they can be easily compared visually.
